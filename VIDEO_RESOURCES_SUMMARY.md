## Quick Start Guide to Recording Your Video

### Step 1: Read the Script (10 min)
Open and read **VIDEO_DEMO_SCRIPT.md** completely

### Step 2: Prepare (15 min)
- Install recording software (OBS Studio/Loom recommended)
- Clean your desktop
- Start the app: `npm run dev`
- Open VS Code with project
- Test audio

### Step 3: Practice (20 min)
- Do one complete run-through without recording
- Practice switching between browser and code
- Time each section

### Step 4: Record (30-60 min)
- Follow VIDEO_DEMO_SCRIPT.md section by section
- Use DEMO_CHECKLIST.md to track features shown
- Record in segments if needed

### Step 5: Review & Upload (30 min)
- Watch your recording
- Edit as needed
- Upload to YouTube/Loom
- Add link to README

## Must-Show Features

Critical (Do Not Skip):
- 3D globe rotation/zoom
- Exchange marker tooltips
- Real-time latency connections with colors
- Click two exchanges ’ see historical chart
- Switch time ranges (1h/24h/7d/30d)
- Search and filter functionality
- Theme toggle (dark/light)
- Mobile responsive view
- Code in VS Code

## Video Quality Settings

- Resolution: 1920x1080 (1080p)
- Frame Rate: 30 fps minimum
- Format: MP4 (H.264)
- Length: 7-10 minutes ideal
